02 — Physics-Informed ML

Landslide Susceptibility Prediction — Custom Dataset & Physics Model

Built a custom multi-source dataset by fusing LULC (Land Use Land Cover), NASA GPM IMERG rainfall, and DEM (Digital Elevation Model) satellite imagery with soil datasets aggregated across platforms. Applied the Infinite Slope Stability model — the same physics framework used by NASA's TRIGRS system and ISRO's NRSC — to derive Factor of Safety (FS) labels:

FS = [c′ + (γz − γwzw) cos²β tanφ′] / [γz sinβ cosβ]
FS > 1 → Stable  |  FS < 1 → Failure risk. Combined with Ensemble ML: Isolation Forest (anomaly detection), Autoencoder + K-Means Clustering (unsupervised pattern extraction), and the physics FS model stacked for final prediction.
